# 🚀 QUICK START — Léame Primero

## Paso 1: Descomprima este ZIP

Guarde la carpeta `perfectamente-coach-starter` donde tenga sus proyectos.
Ejemplo: `C:\Proyectos\` (Windows) o `~/Code/` (Mac/Linux).

---

## Paso 2: Abra en VS Code

1. Abra VS Code
2. **File → Open Folder** → seleccione `perfectamente-coach-starter`
3. Vea estos 5 archivos en el explorer:
   - `README.md`
   - `CONTEXT.md`
   - `SETUP.md`
   - `MENSAJE-PARA-CLAUDE-CODE.md`
   - `QUICK-START.md` (este)

---

## Paso 3: Abra Claude Code

1. Abra la extensión Claude Code en VS Code
2. Inicie un chat nuevo

---

## Paso 4: Adjunte estos archivos al chat

**Arrastre estos archivos al chat de Claude Code** (o use el botón de attach):

✅ `README.md`
✅ `CONTEXT.md`
✅ `SETUP.md`
✅ `references/perfectamente-demo-mobile.html`
✅ `references/perfectamente-coach-landing.html`
✅ `references/plan_extraido.json`

---

## Paso 5: Pegue el mensaje inicial

1. Abra `MENSAJE-PARA-CLAUDE-CODE.md` en VS Code
2. Seleccione TODO (Ctrl+A) y copie (Ctrl+C)
3. Péguelo en el chat de Claude Code
4. Envíe

---

## Paso 6: Responda las preguntas

Claude Code le va a preguntar cosas como:
- ¿Tiene PHP 8.2+ instalado? (Laragon / Herd / XAMPP)
- ¿Tiene Composer instalado?
- ¿Tiene cuenta en GitHub, Anthropic Console, Stripe?
- ¿Su WHM tiene PHP 8.2+ y MySQL 5.7+?

**Si no tiene algo, dígale "no, ayúdame a instalarlo"** — él le da los comandos exactos.

---

## Paso 7: Programen juntos 🥋

A partir de ahí, Claude Code va a guiar paso a paso:
1. Crear el proyecto Laravel
2. Instalar Breeze (auth)
3. Crear migrations según el schema
4. Crear el servicio de Claude API
5. Migrar el demo HTML a Blade
6. Conectar Stripe
7. Deploy a su WHM

---

## ⚡ Reglas importantes que ya están en el contexto

- ✅ Stack: **Laravel 11 + Blade + Alpine.js + Tailwind CDN + MySQL**
- ✅ Hosting: **su WHM/cPanel** (no Vercel)
- ✅ Costo objetivo: **~$1/mes fijo**
- ✅ Tono: **usted, Hormozi-direct**
- ✅ Diseño: **dark + oro #FFD264 + Lora/Inter**
